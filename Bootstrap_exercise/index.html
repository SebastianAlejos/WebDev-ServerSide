My expectations for the class

During this semester I expect to learn about how websites work behind the scenes. Being able to built both the front and back end of a website opens up a world of possibilities. Linking a database with the website allows me to finally be able to produce useful and dynamic web applications that feed on updated data.

I also expect to develop a more extensive portfolio to be able to show to potential employers. I also expect to improve my existing front-end abilities as I practice them when building websites for this class. I hope to learn how to write beautiful and understandable code that meets the expectations for the class.

My Favorite Part of Web Development

My favorite part of web development is realizing that I can build something out of nothing but the ideas that I had in my head. Transforming my ideas into something more real that can have an impact in the real world is what I love about web development.

So far I like the front-end more than the back-end, but I suspect that once I learn more about how the actual programming of websites work I will like it more. I also liike how using Bootstrap makes designing, and creating websites an easier process.
